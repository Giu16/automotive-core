## 📌 Sobre o Projeto

A **Zyron Auto** é uma landing page desenvolvida para uma oficina mecânica especializada com mais de 20 anos de experiência. O foco do projeto foi criar uma interface leve, sem dependências desnecessárias, priorizando performance e carregamento rápido para o usuário final.

O design foi estruturado para conversão: os clientes podem facilmente conhecer a infraestrutura, conferir os diferenciais, ler as avaliações reais e entrar em contato para agendar uma avaliação.

## 🚀 Funcionalidades

- **Catálogo de Serviços:** Apresentação de soluções (Revisão, Freios, Motor, Suspensão, Elétrica e Ar Condicionado).
- **Social Proof:** Integração com layout de avaliações do Google (438 reviews) para gerar autoridade imediata.
- **Diferenciais Competitivos:** Destaque para técnicos certificados, peças originais e garantia total.
- **Seção de FAQ:** Respostas rápidas em formato sanfona para quebrar objeções comuns dos clientes.
- **Design Responsivo:** A interface se adapta perfeitamente a dispositivos móveis e desktops.
